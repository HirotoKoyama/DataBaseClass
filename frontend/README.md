# README.md

## 必要なもの

- npm
  - Node.jsに同梱されてるので、node.jsをインストールするなりする。

## 動作確認の手順

1. `npm install`を実行する
2. 好きなファイルを見る

## ファイルと概要

- `index.html`: TOPページ。テーブルやページネーションは適宜バックエンドに合わせて置き換える
- `login.html`: ログインページ。`input`要素の`name`属性とかは適宜書き換える
- `editor.html`: エディタページ。右側の背景黒の部分にターミナルを入れる。メニューや解答の内容もバックエンドの内容を入れる場所。
